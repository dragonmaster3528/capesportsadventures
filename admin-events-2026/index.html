How to Manage Events

  1. Select a sport tab to view its events
  2. Click "Edit" to modify an event, or "Delete" to remove it
  3. Click "Add New Event" to create a new event for the selected sport
  4. When done, click "Download JSON" and upload to /data/ folder on Netlify
